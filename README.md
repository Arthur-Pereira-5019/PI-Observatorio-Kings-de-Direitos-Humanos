
# Observatório Kings de Direitos Humanos do Vale do Itajaí 

REF https://github.com/role-pi/app

O projeto Observatório Kings de Direitos Humanos do Vale do Itajaí é um observatório de direitos humanso como o nome diz que busca democratizar o acesso a informação de grupos socialmente marginalizados no vale do Itajaí.

Buscamos também Permitir especialistas cadastrarem infográficos e pesquisas sobre os direitos de populações marginalizadas no Vale do Itajaí.

Agregar notícias divulgadas no Vale do Itajaí relevantes às populações marginalizadas.

Divulgar pessoas, espaços e instituições que possam ser de utilidade para minorias.

Criar um espaço acolhedor para as minorias marginalizadas do Vale do Itajaí.
